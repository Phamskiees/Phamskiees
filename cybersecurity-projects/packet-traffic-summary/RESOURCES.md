# Resources

- [TShark manual](https://www.wireshark.org/docs/man-pages/tshark.html) -
  authoritative syntax for reading captures and exporting selected fields.
- [Wireshark User's Guide](https://www.wireshark.org/docs/wsug_html_chunked/) -
  packet navigation, display filters, and analysis workflow.
- [Wireshark Display Filter Reference](https://www.wireshark.org/docs/dfref/) -
  searchable protocol and field names.

Study connection: TShark uses the same dissectors and display-filter language
as Wireshark, making it suitable for turning manual lab observations into a
repeatable defensive-analysis workflow.
