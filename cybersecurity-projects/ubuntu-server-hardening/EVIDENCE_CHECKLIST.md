## Public Safety Notes

Before publishing screenshots, redact:

- personal usernames;
- private lab IP addresses if they are not needed for the lesson;
- MAC addresses;
- browser profile details;
- local filesystem paths;
- passwords, keys, tokens, and recovery codes.
